# Лабораторные работы по предмету "Алгоритмизация и программирование"

Репозиторий содержит набор лабораторных работ, выполненных в рамках университетской практики по предмету "Алгоритмизация и программирование". Работы написаны на языке C и демонстрируют основные концепции программирования: указатели, массивы, структуры, рекурсия, работа с файлами и тестирование.

## Структура проекта

- **recursion.c**: Дополнительное задание на понимание рекурсий. 
- **lab1/**: Лабораторная работа 1.
  - lab1.c: Основной файл.
- **lab2/**: Лабораторная работа 2.
  - 1.h, 2.h: Заголовочные файлы.
  - lab2_1to4.c, lab2_part2.c: Реализации заданий.
  - useful_constants.h: Константы.
- **lab3/**: Лабораторная работа 3.
  - lab3.c: Основной файл.
- **lab4/**: Лабораторная работа 4 (Указатели, массивы, рекурсия, функции с переменным числом параметров и тестирование).
  - lab4.c: Основной файл.
  - functions.c: Реализации функций.
  - functions.h: Заголовки функций.
  - tests.c: Тесты.
  - minunit.h: Фреймворк для тестирования MinUnit.
  - README.md: Подробное описание лабораторной работы 4.
- **lab5/**: Лабораторная работа 5.
  - lab5.c: Основной файл.
- **lab6/**: Лабораторная работа 6 (Структуры, массивы структур, динамическое выделение памяти, работа с файлами и сортировка).
  - lab6.c: Основной файл.
  - functions.c: Реализации функций.
  - functions.h: Заголовки функций.
  - book.h: Определение структуры Book.
  - catalog.txt: Файл для хранения картотеки (создается автоматически).
  - README.md: Подробное описание лабораторной работы 6.
- **.gitignore**: Файл для игнорирования ненужных файлов в Git.

## Лабораторные работы

### Лабораторная работа 1
Базовые концепции программирования на C.

### Лабораторная работа 2
Работа с заголовочными файлами и модульной организацией кода.

### Лабораторная работа 3
Продвинутые темы в C.

### Лабораторная работа 4
Указатели, массивы, рекурсия, функции с переменным числом параметров и тестирование. Подробности см. в [lab4/README.md](lab4/README.md).

### Лабораторная работа 5
Структуры и базовые операции с ними.

### Лабораторная работа 6
Картотека книг с использованием структур, динамической памяти, файлов и сортировки. Подробности см. в [lab6/README.md](lab6/README.md).

## Компиляция и запуск

Каждая лабораторная работа компилируется отдельно. Общий пример для GCC:

```bash
gcc -o labX labX/labX.c [дополнительные файлы]
./labX
```

Для работ с несколькими файлами (например, lab4 и lab6) следуйте инструкциям в соответствующих README.md.
