# Лабораторная работа 6

Лабораторная работа посвящена структурам, массивам структур, динамическому выделению памяти, работе с файлами и сортировке в языке C.

## Структура проекта

- **lab6.c**: Основной файл с функцией main
- **functions.c**: Реализации всех функций для работы с картотекой книг
- **book.h**: Заголовочный файл с определением структуры Book
- **functions.h**: Заголовочный файл с заголовками функций и перечислением SortingType
- **catalog.txt**: Файл для сохранения/загрузки картотеки (создается автоматически)

## Компиляция и запуск

```bash
gcc -o lab6 lab6.c functions.c
./lab6
```

## Задания

### Задание 6.1. Картотека книг
Создайте «картотеку» книг на основе структуры Book из ЛР5. Картотека реализована как динамический массив структур, размер которого увеличивается по мере добавления книг (начальная емкость 10, удваивается при необходимости).

Пользователь может выполнять следующие действия через меню:
- Распечатать содержимое картотеки
- Ввести новую книгу
- Удалить существующую книгу
- Записать картотеку в файл
- Считать картотеку из файла
- Выход из программы

### Задание 6.2. Сортировка картотеки
Реализуйте возможность сортировки массива книг по любому из полей структуры. Признак поля сортировки задается с помощью перечисления SortingType.

Доступные поля для сортировки:
- Автор
- Название
- Год
- Цена
- Категория

## Функции

- `printBook()` - вывод информации о книге
- `printCatalog()` - вывод всей картотеки
- `addBook()` - добавление книги с автоматическим увеличением массива
- `removeBook()` - удаление книги по индексу
- `saveCatalogToFile()` - сохранение в файл
- `loadCatalogFromFile()` - загрузка из файла
- `sortCatalog()` - сортировка по выбранному полю
- `freeCatalog()` - освобождение памяти
